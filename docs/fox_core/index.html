fox_core

Players, characters, accounts and vehicles/horses for RedM servers.

fox_core manages sessions, characters, groups, statuses, accounts (money/gold) and vehicles — including horses, which on RDR3 are peds, not vehicle.

Installation

Import the SQL in fox_core/sql/install.sql into your database before starting the resource.

Exports

The API mirrors ox_core (same method names: GetPlayer, CreateVehicle, etc.) — only the resource name changes.
