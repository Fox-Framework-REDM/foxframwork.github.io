OxAccount (Server)

Money/gold accounts — personal, group or shared.

OxAccount represents a bank account (personal, group or shared). Obtained via OxAccount.get(accountId) or the GetCharacterAccount/GetGroupAccount exports.

Money and gold are floats

money/gold are stored as cents (float). Deposit/withdraw methods receive the amount in dollars; the conversion to inventory storage happens internally.
