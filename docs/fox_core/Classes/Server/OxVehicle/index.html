OxVehicle (Server)

Vehicles, carts and horses (RDR3 treats horses as peds, not vehicles).

OxVehicle represents any persisted vehicular entity — wagons, carts, boats and horses. On RDR3, horses are peds, so spawn() branches internally to CreatePed when type === 'horse' in vehicles.json, and to CreateVehicleServerSetter for everything else.
