OxPlayer (Server)

Main server-side player class — session, character, groups, statuses and licenses.

OxPlayer represents a connected player's session. It's obtained via OxPlayer.get(source) or the GetPlayer export.

Health on RDR3

GetEntityHealth is unreliable server-side for player peds on RDR3. The client replicates health through the health statebag (see States); getHealth() only reads that value, it never calls the native directly.
