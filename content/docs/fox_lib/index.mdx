---
title: fox_lib
description: Small RedM utility library for Fox resources.
---

`fox_lib` provides reusable RedM helpers for entities, NPCs, blips, props, cameras, placement, target adapters, security, and weapon UX.

## Requirements

- `ox_lib`
- `ox_inventory` only for weapon oil / weapon metadata integrations
