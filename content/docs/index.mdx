`fox_core` is a framework for **RedM** (Red Dead Redemption 2 / `rdr3`), built on the architecture of
[overextended](https://github.com/overextended)'s `ox_core`, but natively ported to RDR3's
natives and limitations — no cars, armour, plates or phones; horses, cores, and telegrams instead.

## Where to start

- [fox_core](/docs/fox_core) — players, characters, accounts and vehicles/horses.

## Conventions

- Network events keep the `ox:` namespace for contract stability, even though the resource is `fox_core`.
- Exports are accessed via `exports.fox_core` (`Ox = exports.fox_core`).
- Convars use the `ox:` prefix (e.g. `ox:spawnModel`).
